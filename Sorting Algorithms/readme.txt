Bubble Sort
An algorithm that repeatedly compares adjacent elements and swaps them if they are in the wrong order, gradually moving the larger elements to the end of the array.

Insertion Sort
An algorithm that builds the sorted array one element at a time by inserting each element into its correct position among the previously sorted elements.

Selection Sort
An algorithm that repeatedly finds the smallest element in the unsorted portion of the array and moves it to the correct position at the beginning of the array.
