<h1>Project Overview</h1>
This project uses different kind of algorithms to sort numerical arrays

<h2>How to use</h2>
In functions.py you can see all libraries that contain the sorting algorithms and a function call. You can change what algorithm you want to use, specifying library name + the correct function

<h3>How does it works</h3>
Every algorithm inside the project will receive one array and so it'll be sorted.

<h3>Main goal of the project</h3>
The goal of this project was to learn about sorting algorithms, algorithmic complexity, and good programming practices
